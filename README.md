# Wedding Invitation

A beautiful, interactive, and responsive digital wedding invitation website. This project features a stunning envelope opening animation, a custom music player, floating particles, scroll animations, and an RSVP form.

## 🌟 Live Preview

Check out the live invitation here:
[Live Website](https://akashdavidkumar.github.io/Wedding_Invitation/)

## ✨ Features

- **Interactive Envelope:** A beautifully animated digital envelope that opens to reveal the invitation details.
- **Custom Cursor & Particles:** A personalized cursor trail and animated background particles for a magical feel.
- **Background Music Player:** An integrated, stylish music player with a custom vinyl record animation that plays romantic background music.
- **Scroll Animations:** Smooth reveal animations for sections as the user scrolls down the page.
- **Responsive Design:** Optimized to look perfect on devices of all sizes, from mobile phones to desktop monitors.
- **Event Timeline:** Clear and elegant presentation of wedding events (Sangeet, Wedding, Reception).
- **RSVP Form:** A built-in modal for guests to easily RSVP and leave a message.
- **Gallery & Countdown:** Includes a photo gallery and a live countdown timer to the big day.

## 🛠️ Technologies Used

- **HTML5:** For structuring the content and semantic elements.
- **CSS3 (Vanilla):** For all styling, complex animations (keyframes, transitions), responsive layouts (Flexbox/Grid), and glassmorphism effects.
- **JavaScript (Vanilla):** For interactive elements (envelope toggle, music player logic, modal handling, particle canvas, scroll reveals).
- **FontAwesome:** For various icons used throughout the site.
- **Google Fonts:** Utilizing 'Dancing Script', 'Playfair Display', and 'Poppins' for elegant typography.

## 🚀 Getting Started

To view the project locally:

1. Clone this repository:
   ```sh
   git clone https://github.com/AkashDavidKumar/Wedding_Invitation.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Wedding_Invitation
   ```
3. Open `index.html` in your preferred web browser.

## 📁 Project Structure

- `index.html`: The main and only HTML file containing the structure, embedded CSS, and JavaScript.
- `assets/`: Directory containing images, music files, and other required assets.
